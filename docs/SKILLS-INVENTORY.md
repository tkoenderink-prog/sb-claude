All 34 skills follow Claude Agent SDK SKILL.md format:

**Frontmatter (YAML)**:
```yaml
---
name: skill-slug
description: "Brief description"
trigger: "when user mentions: keywords, phrases"
personas: [persona-slugs] or [all]
---
```

**Structure**:
1. Overview section
2. When to use this skill
3. Persona approaches (for multi-persona skills)
4. Skill logic / execution flow
5. Example usage
6. Quality checks / testing
7. Notes / evidence sources
